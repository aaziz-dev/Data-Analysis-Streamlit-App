# Data-Analysis-Streamlit-App
# ☕ Beans & Pods – Data Analysis & Streamlit App

## 📌 Overview

Beans & Pods is a fast-growing coffee company looking to better understand its sales performance in order to increase revenue.

This project focuses on analyzing sales data, identifying trends, and providing actionable business recommendations using data science techniques.

---

## ⚙️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib / Seaborn
* Scikit-learn (optional)

---

## 🚀 Features

* 📊 Interactive dashboard using Streamlit
* 📈 Sales data analysis (trends, patterns)
* 🔍 Insights on top-performing products and regions
* 📉 Visualization of revenue distribution
* 💡 Business recommendations for marketing strategy

---

## 📊 Key Insights

* Identification of the most profitable products
* Analysis of customer behavior trends
* Detection of high-revenue regions
* Seasonal or time-based sales patterns

---

## 💡 Recommendations

* Invest in high-performing products
* Target marketing campaigns based on top regions
* Optimize pricing strategies
* Focus on customer segments generating the most revenue

---

## 🧠 What I Learned

* Data analysis and visualization
* Building interactive dashboards with Streamlit
* Extracting business insights from raw data
* Communicating results to non-technical stakeholders

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run etude_cafe.py
```
